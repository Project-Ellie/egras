# Support Impersonation

Operator can act on behalf of a user for support, with explicit audit trail and (ideally) user consent.

**Scope:** act_as JWT claim, audit events tagged with both principal and impersonator, optional time-boxed user-granted token.

**Touches:** [[Authentication]], [[Audit-System]].
