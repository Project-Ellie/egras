# Data Residency

Tenant data physically stays in chosen region (EU, US, CH).

**Scope:** Region tag per org, routing layer, per-region DB instances. Cross-region admin tooling.

**Touches:** [[Architecture]], deployment topology.

**Why maybe later:** Huge operational lift; only justified by EU/CH regulated customers.
