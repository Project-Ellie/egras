# User & Org Inbox

Durable in-app message store per user and per org. (Wolfgang named this.)

**Scope:** Message thread, sender (system or user), read/unread, retention. REST + websocket/SSE push. Org inbox visible to org members per permission.

**Touches:** new domain messaging/. Pairs with [[Notification-Channels]].
