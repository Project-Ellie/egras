# Notification Channels

Fan-out a single notification event to email, webhook, in-app inbox per user preference.

**Scope:** Channel registry, user preference matrix, delivery receipts, retry on failure.

**Touches:** pairs with [[User-and-Org-Inbox]], [[Webhook-Subscriptions]].
